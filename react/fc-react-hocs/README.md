# fc-react-hocs

fc-react-hocs 是 React 开发中一系列常用高阶组件的集合。

- 源代码查看: [code/src/hoc](../code/src/hoc)
- 示例代码查看: [code/stories/hoc](../code/stories/hoc.js)

# Usage

```sh
# 在 JavaScript 项目中使用
$ create-react-app myapp

# 在 TypeScript 项目中使用
$ create-react-app myapp --scripts-version=react-scripts-ts-antd
```
